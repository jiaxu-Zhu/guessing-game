# 🎯 猜谜大师 - 精美猜谜游戏

一个功能丰富、界面精美的网页猜谜游戏，包含多种游戏模式和丰富的交互效果。

## ✨ 功能特性

### 🎮 四种游戏模式
1. **数字猜猜乐** - 猜 1-100 之间的数字
2. **词语猜猜猜** - 根据提示猜词语
3. **动物猜猜看** - 猜动物名称
4. **表情猜成语** - 通过 emoji 猜成语

### 🎨 精美界面
- 渐变背景 + 动态粒子效果
- 流畅的动画和过渡效果
- 响应式设计，支持手机和电脑
- 现代化的卡片式布局

### 📊 游戏功能
- ⏱️ 实时计时器
- 💡 提示系统（每局最多3次）
- 📈 进度追踪
- 📝 猜测历史记录
- 🏆 成就系统
- 📱 快速数字按钮（数字模式）

### 📈 统计数据
- 总游戏次数
- 获胜次数
- 胜率统计
- 最高连胜记录
- 数据本地持久化

## 🚀 快速开始

### 方法一：直接打开
1. 下载所有文件到同一文件夹
2. 用浏览器打开 `index.html`

### 方法二：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .
```
然后访问 `http://localhost:8000`

## 🎯 游戏玩法

1. **选择模式** - 在主菜单选择喜欢的游戏模式
2. **开始猜测** - 根据提示输入你的答案
3. **使用提示** - 遇到困难时可以使用提示（每局3次）
4. **查看历史** - 实时查看之前的猜测记录
5. **挑战高分** - 尽量用更少的时间和次数获胜

## 🏆 成就系统

- **连胜大师** - 连续获胜达到一定次数
- 更多成就持续解锁中...

## 📱 移动端支持

游戏完全支持手机和平板设备：
- 触摸友好的按钮
- 响应式布局
- 优化的字体大小
- 流畅的移动端体验

- 无需任何外部依赖

## 🎨 自定义

### 添加新词语（词语模式）
编辑 `game.js` 中的 `words` 数组：
```javascript
words: [
    { word: "太阳", hint: "每天从东方升起..." },
    // 添加更多...
]
```

### 添加新动物（动物模式）
编辑 `game.js` 中的 `animals` 数组：
```javascript
animals: [
    { name: "狮子", hints: ["草原之王", "有鬃毛", "猫科动物"] },
    // 添加更多...
]
```

### 添加新成语（表情模式）
编辑 `game.js` 中的 `idioms` 数组：
```javascript
idioms: [
    { idiom: "画蛇添足", emoji: "🎨🐍➕🦶" },
    // 添加更多...
]
```
